### Zalando — Senior Data Engineer (Partner Tech)
**Location**: Berlin, Germany
**Source**: jobs.zalando.com
**URL**: https://jobs.zalando.com/en/jobs/2723400

#### What they ask for:
- Apache Spark (PySpark) and Python for large-scale data processing
- Several years building and maintaining scalable Streaming & Batch pipelines
- CDC (Change Data Capture) from RDBMS (e.g., MySQL / BinLogs)
- Infrastructure as Code (Terraform), Docker, Git (trunk-based development), DevOps principles
- Complex business requirements → de-normalised data models optimised for Lakehouse performance
- Cloud Infrastructure (AWS), REST APIs, secure data exchange
- Databricks experience — described as a **must**
- Airflow for orchestration
- MPP warehouses: Redshift, BigQuery, Snowflake
- English communication: articulate complex technical topics concisely

#### Nice to have:
- Experience technically guiding and leading data engineers and analysts
- Aligning data strategies within a larger group ecosystem

#### CV Keywords to Mirror:
`PySpark`, `Apache Spark`, `CDC`, `Change Data Capture`, `Terraform`, `Databricks`, `Delta Lake`,
`Airflow`, `AWS`, `Lakehouse`, `batch pipelines`, `streaming pipelines`, `data governance`

#### What the interview looks like (from Glassdoor):
2hr Codility test (SQL + Spark coding + general coding) → technical discussion with engineering lead → system design → culture fit. Expect live Spark coding, SQL problems, and system design scenarios.

## Skills Frequency Analysis Across All Postings

Based on the above real postings, here are the most frequently required skills by category:

- Python
- SQL
- Apache Spark / PySpark
- Cloud (AWS, Azure, GCP — at least one)
- Git + version control
- CI/CD
- Data pipelines (batch and/or streaming)

### AI/ML roles specifically (3+ postings):
- PyTorch or TensorFlow
- LLMs + RAG (Retrieval-Augmented Generation)
- Model deployment and monitoring
- Docker / Kubernetes
- Vector databases / embeddings

### Growing fast (appears in 2+ postings, trending up):
- Agentic AI / AI agents
- MLOps
- Databricks
- dbt
- Causal inference
- A/B testing
- Azure OpenAI API

### What European companies care about that US companies don't emphasise as much:
- English written and verbal communication (explicitly required, especially non-English HQs)
- GDPR awareness / data privacy (Booking.com, Zalando, Klarna all mention it)
- Cross-functional collaboration described in detail (not assumed)
- Relocation support mentioned at Klarna for Stockholm

|---|---|---|---|

| Data Scientist | €50–65k | €70–95k | €90–130k |

**Notes**:
- Germany (Munich/Berlin) and Netherlands (Amsterdam) pay 10–20% above Iberia

- AI skills (LLMs, RAG, agents) command 25–40% premium over general ML engineers
